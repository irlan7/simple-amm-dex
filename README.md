# Uniswap-Style AMM (Constant Product)
A decentralized exchange (DEX) logic implementing the constant product formula (x * y = k).

## Features
- Token Swaps with a 0.3% fee.
- Liquidity Provision (Add & Remove Liquidity).
- Share calculation and reserve updates mirroring Uniswap V2 math.
- Built with Foundry & OpenZeppelin.
